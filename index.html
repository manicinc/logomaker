About Performance

Initial loading might be slower on GitHub Pages than when run locally. This is by design:

  • GitHub Pages has issues parsing font files from LFS. As such..
  • All 400+ fonts are embedded directly as Base64 data
  • This also enables our unique "one-file" approach - no server required!
  • You can download this entire app as a single HTML file and run it offline
  • Share it via email, USB drive, or any way you want - it just works

Once loaded, everything runs smoothly in your browser. This extreme portability is what makes Logomaker special!
