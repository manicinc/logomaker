‚ö° On Logomaker Performance & Loading ‚ö°

  • Optimized Loading: This version uses font chunking for faster initial loads (dist/github-pages) or local servers.
  • Only a small index loads first, then full font data (~400 fonts total) is fetched in small chunks as needed.
  • Portable Mode: The portable build (dist/portable/logomaker-portable.html) embeds all font data (~90MB).
  • This dual approach provides both speed (online chunking) and guaranteed offline operation (portable)!
  • The site currently has some bugs with rendering, for PNGs with text gradients, and issues rendering SVGs. Fixes coming soon.
  • Source Code on GitHub: Logomaker
