About Logomaker ‚ö°

    You might've experienced the site being a bit slow to load..

    Logomaker allows you to easily manage fonts (BYOF), including being able to embed fonts directly in the app for offline / single-file usage.

  • (Over 400 fonts are included directly in this deployment's data, see why below.)
  • This seems to be a limitation with fonts loading from LFS and GitHub pages. But we've also made use of it to enable bundling the app into just one (giant) HTML file, that can be run without a server!
  • Initial load will take a long time as there's no lazy or optimal loading of fonts yet.
  • Run downloaded HTML offline, share via USB - it just works! Bundled down with PortaPack.
  • Download here: https://github.com/manicinc/logomaker
  • Logomaker is open-source and forever free hosted on GitHub pages!

Y'all got them vector animations? ü§î

    Yes! When you export as SVG, active CSS animations (Pulse, Float, etc.) are included within the SVG file using style tags and @keyframes.

    Browsers supporting CSS animations in SVGs will play the animation when you open the SVG file directly.

    (Note: Compatibility may vary depending on the SVG viewer/browser.)

Future Roadmap üö™

    Logomaker started out as an internal tool that Manic.agency decided to release as a fun side project to our library PortaPack.

    While Logomaker will continue to be free and open-source forever as well as permanently hosted here on GitHub pages, further development is unplanned though PRs are welcome.

    At Manic, we are extending Logomaker for our own imaging and design purposes, and plan to make the full platform available for users in the future. Check back at Manic to stay tuned!

About Logomaker üé®

                  
Logomaker is a free, open-source tool by Manic Agency for creating text logos. It emphasizes portability via embedded fonts.

                  
Features: Styling, gradients, effects, borders, animations, backgrounds, PNG/SVG/Frames export.

                  
Source: GitHub (MIT License)
